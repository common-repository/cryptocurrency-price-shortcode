== Description ==
Crypto Price Shortcode Plugin is a plugin that helps you to submit the price of different cryptocurrencies in your site with only shortcodes!
You are also able to show the price of cryptocurrencies with the fiats you want!
Fiat currencies supported: "AUD", "BRL", "CAD", "CHF", "CLP", "CNY", "CZK", "DKK", "EUR", "GBP", "HKD", "HUF", "IDR", "ILS", "INR", "JPY", "KRW", "MXN", "MYR", "NOK", "NZD", "PHP", "PKR", "PLN", "RUB", "SEK", "SGD", "THB", "TRY", "TWD", "ZAR"
*This plugin uses coinmarketcap.com API.
== Installation ==

The installation is very easy!

1. Upload the plugin in wordpress
2. Activate the plugin
3. Use shortcode [price id="the_name_of_cryptocurrency_you_want" fiat="currency"] - For example [price id="bitcoin" fiat="usd"]
